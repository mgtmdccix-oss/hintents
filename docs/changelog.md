# Project Changelog

Welcome to the official ERST changelog. Here we document all major updates and how they impact your development workflow.

## [Unreleased]

### [NEW] New Feature: Time-Travel Debugging
We are currently building the foundation for **Time-Travel Debugging**. This will allow you to:
- **Rewind** execution to any host function call.
- **Inspect** ledger state diffs at every step.
- **Debug** complex traps by seeing exactly what changed before the crash.

**Impact**: High-fidelity debugging will become significantly faster and more intuitive.

---

## [v2.0.0] - 2026-03-20

### [DEPLOY] Major Improvements
- **Interactive Trace Viewer**: A full TUI for exploring transaction traces.
- **Performance Profiling**: Generate flamegraphs to find bottlenecks in your contracts.
- **Audit Log Signing**: Sign your results with hardware wallets or HSMs.

**Impact**: This release transforms ERST from a CLI tool into a comprehensive developer diagnostic suite.
